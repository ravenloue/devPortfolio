About Me

Projects

  • Smart Clinic CMS

    A full-stack content management system for a fictional hospital to streamline operations for patients, doctors, and administrators.
    Built using Spring Boot, PostgreSQL, & Thymeleaf/JS, the application supports role-based dashboards, JWT authentication, appointment booking workflows, and stored functions for generating real-time insights into hospital performance.
    Deployed using Docker & Render.

    Java Spring Postgres

  • Dealership Review Site

    A full-stack web application for a national car dealership network to centralize customer reviews from branches across the United States.
    Built using Django for backend logic and data modeling, with BootStrap and React for a responsive, interactive front-end.
    Supports user authentication, role-based access, and real-time review management.
    Deployed using Docker & Render.

    Python Django React

  • Nursery Shopping App

    A front-end e-commerce web application for an online plant shop built using React, Redux, and React Router, featuring a dynamic product catalog, category-based listings, and a state- managed shopping cart with real-time updates.
    Implements reusable UI components, global state management for cart actions, and seamless navigation between product and checkout pages.

    React Redux React Router

  • TMDB Trending Search

    A mobile-friendly web application that integrates with The Movie Database (TMDB) API to display daily and weekly trending movies, TV shows, and industry personalities.
    Built using TypeScript and Bootstrap 5, the app dynamically fetches and renders content with clean, uniform cards, and responsive layout design. Users can filter results by category and timeframe for a tailored browsing experience.
    Deployed via Github Pages.

    TypeScript Bootstrap JSON

  • Simple Bank

    A PHP-based mock banking web application that simulates basic banking operations through a modular, object-oriented structure inspired by modern frameworks like React.
    Built with PHP 8, HTML5, CSS3, and JavaScript, this project emphasizes clean UI design, component reusability, and database interaction best practices.
    Deployed using Epizy as a way to explore full-stack workflows from concept to development.

    PHP JavaScript MySQL
